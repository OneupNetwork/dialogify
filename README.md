# Dialogify
A javascript plugin for creating dialog/lightbox, implements with HTMLDialogElement.

## Basic usage
```javascript
new Dialogify('dialog content')
    .title('dialog title')
    .buttons([{type:Dialogify.BUTTON_PRIMARY}])
    .showModal();
```
